# Phases Reference

Use this as the short version of [docs/ROADMAP.md](../../../docs/ROADMAP.md) when working through the skill.

## Phase A

Scaffold and contracts only. No runtime feature work unless it is required to prove an interface.

## Phase B

Define and implement the minimal channel-agnostic core runtime slice.

## Phase C

Deliver adapter integrations against the shared contract. Telegram was first; OpenClaw is now the primary runtime invocation path.

## Phase D

Add operational hardening and public-release workflows (issues, releases, maintenance docs) while preserving adapter/core boundaries.
